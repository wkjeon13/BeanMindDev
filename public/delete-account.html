BeanMind 사용자 데이터 삭제 안내

BeanMind는 사용자의 개인정보와 데이터 삭제 권리를 존중합니다. BeanMind 앱을 통해 수집되거나 저장된 사용자 데이터의 삭제를 요청하려면 아래 절차를 따라주세요.

1. 삭제 요청 방법

사용자 데이터 삭제를 요청하려면 아래 이메일 주소로 요청을 보내주세요.

이메일: wkjeon@gmail.com

2. 이메일에 포함할 정보

  • 이름 또는 앱에서 사용한 계정 이름
  • 로그인에 사용한 이메일 주소
  • “BeanMind 사용자 데이터 삭제 요청” 문구

3. 삭제되는 데이터

  • BeanMind 앱 계정 정보
  • 앱 사용 과정에서 저장된 사용자 관련 데이터
  • 로그인 또는 연동을 통해 수집된 식별 정보

4. 처리 기간

삭제 요청은 확인 후 영업일 기준 7일 이내에 처리됩니다. 법적 의무 또는 보안상의 이유로 일부 기록은 관련 법령에 따라 제한된 기간 동안 보관될 수 있습니다.

5. 문의

데이터 삭제와 관련한 문의는 wkjeon@gmail.com 으로 연락해주세요.
